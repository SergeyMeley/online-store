# OnlineStore - Интернет-магазин на ASP.NET Core

![ASP.NET Core](https://img.shields.io/badge/ASP.NET_Core-6.0-blue)
![Bootstrap](https://img.shields.io/badge/UI-Bootstrap%205-purple)

Полнофункциональный интернет-магазин с каталогом товаров и отзывами пользователей.

## Основные возможности

- Каталог товаров с фильтрацией по наименованию, категории и цене
- Система оценок и отзывов
- Адаптивный дизайн (Bootstrap 5)

## Технологический стек

### Backend
- ASP.NET Core 6.0
- Entity Framework Core
- ASP.NET Core Identity

### Frontend
- Bootstrap 5
- Razor Pages

### База данных
- PostgreSQL
